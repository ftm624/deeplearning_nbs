Deep Learning Notebooks¶

An exploration of deep learning from the ground up.

These notebooks are loosely based on the Fastai course Part 2: Deep Learning from the Foundations . They follow the general order of that course and expand upon cencepts and code I found to be particularly interesting.

This documentation is not meant to build a functional library for deep learning. Instead it is an attempt at "literate programming" (Knuth, 1983).
